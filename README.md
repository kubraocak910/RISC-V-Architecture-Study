# RISC-V-Architecture-Study
Bilgisayar mimarisi prensipleri, RISC-V mimarisi ve işlemci performansı üzerine çalışmalar.

RISC-V Assembly: Bellek yönetimi ve fonksiyon çağrıları optimize edildi.

Performance Analysis: İşlemci çevrim süreleri (CPI) ve yürütme zamanı üzerine matematiksel modeller kuruldu.

Memory Hierarchy: Önbellek (Cache) ve Sanal Bellek yapılarının performans üzerindeki etkileri analiz edildi.
